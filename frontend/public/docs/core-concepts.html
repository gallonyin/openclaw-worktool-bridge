核心概念

机器人

WorkTool 里的一个机器人实体。每个机器人有唯一 robot_id。

规则

规则定义“什么消息该由谁回答”。

常见维度：

  • 来源场景：群聊 / 私聊
  • 对象匹配：全部 / 精准 / 模糊
  • 内容匹配：全部 / 精准 / 模糊
  • 权重：同类命中时的优先级

AI 回复引擎

一条可调用的大模型配置，包括：

  • Base URL
  • API Token
  • Model
  • 引擎类型

回调

WorkTool 把收到的消息推送给本系统的入口。

如果绑定的是本平台回调地址，本系统可记录更多本地处理信息。

消息转发

把机器人收到的文本消息转发到另一个目标（群名或用户昵称/备注名）。
