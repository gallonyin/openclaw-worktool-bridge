# 机器人与回调配置

## 添加机器人

在“机器人配置”中点击“添加机器人”。

建议：

- 初期先开启私聊回复
- 群聊先在测试群验证后再放开

## 回调配置建议

在“机器人信息”页，优先使用“由本平台处理回复消息”。

优点：

- 无需自己开发回调服务
- 本平台可记录处理耗时和失败原因
- 小白用户可快速跑通全链路

## 什么时候需要自定义回调

你已有自建系统，且希望 WorkTool 消息同时进入你自己的服务。

这时可以手工填写回调地址。

## 常见问题

- 保存失败：检查地址是否可访问、证书是否正常
- 收不到消息：检查 WorkTool 侧是否确实绑定到目标地址
